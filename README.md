# Resume & Portfolio — Comic-style responsive template

Quick static site for a resume/portfolio with comic-book style transitions and halftone texture.

Files added
- `index.html` — main page with sections (hero, about, projects, resume, contact)
- `styles.css` — responsive styles, comic look, halftone overlay, animations
- `script.js` — IntersectionObserver-based show/hide, keyboard navigation, small parallax

How to use
1. Open `index.html` in your browser (double-click or right-click -> Open With).
2. Scroll or use arrow keys / j / k to navigate between sections.

Notes and next steps
- Replace placeholder content (Your Name, project text, links) with your real info.
- To export as a single-page PDF or to add a real resume PDF, link the download anchor to your file.
- Accessibility: landmarks and a skip link are included; consider adding more semantic markup and focus styles if needed.

Customization ideas
- Swap the font (Bangers) for another comic font or your brand font.
- Add images/screenshots to project cards and link them to detail pages.
- Tweak the halftone density in `styles.css` for a stronger or subtler texture.
